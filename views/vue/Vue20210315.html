    # v-bind基本用法

    • 许多属性需要通过服务器读取，例如a元素的href属性、img元素的src属性

    • 语法糖/缩写为一个：

<a v-bind：src="aHref"></a>   //使用正确
<a ：src="aHref"></a>   //使用正确

aHref：'http://www.baidu.com'

    # 参数问题：

    • 事件调用的方法没有参数时,一下两者使用相同 @click="btnClick()"

      @click="btnClick"

    • 若事件方法需要参数，但省略的小括号，那么vue会默认将浏览器生产的event事件对象作为参数传入

    • 若事件方法又需要自定义参数，又需要event对象，则使用$event

      @click="btnClick(x,$event)"

    # v-show

    作用：也是用与元素是否显示

    与v-if的区别：

    • v-if：当条件为false时，对应的元素不会存在DOM中
    • v-show：当条件为false时，仅仅将元素的display属性设置为none

    选择：

    • 当显示与隐藏切换频繁时，使用v-show
    • 当只有一次切换时，使用v-if

    # 数组相应操作

    通常是对遍历之后的数组进行的响应式操作

    • .push(内容): 在数组末尾添加一个元素

    • .pop()：删除数组中的最后一个元素

    • .shift():删除数组中的第一个元素

    • .unshift(内容)：在数组最前面添加元素，可添加多个

    • .splice()：删除元素/插入元素/替换元素 第一个元素表示起始位置，第二个元素表示要删除几个元素，第三个元素开始就是插入的内容

      .splice(1,3,'a','b','c',d)  表示在1位置开始删除三个元素并插入四个元素
      1

    • .sort():数组排序

    • .reverse:数组倒序

    注意

    通过数组索引修改元素不是响应式的

    # 修饰符

    • lazy修饰符
      • 默认情况下v-model是同步input的数据的，自动发生改变
      • lazy修饰符可以让数据在失去焦点或者回车时才更新
    • number修饰符
      • 默认情况下，在输入框中输入的都会被默认成为字符串类型处理，不管是数字还是字母
      • number修饰符可以让输入框中输入的内容自动转换成数字类型
    • trim修饰符
      • trim修饰符可以过滤内容左右两边的空格

    # 父组件和子组件

    # 组件获取Vue实例数据

    组件是封装的，是不能直接通过{{}}访问到Vue实例中的data

    组件有属于自己的HTML模块，也应该有属于自己的数据data

    但组件的data属性必须是一个函数，通过返回对象数据

    用函数的好处是多次使用组件出来的对象是不同的 互不影响

    <template id='myCpn'>
  <div>  <h2> {{message}} </h2>    </div>

    # 子传父

    具体过程：

    先给子组件通过@click绑定一个事件

    这个事件要用到this.$emit(传递的事件名，传递的值)

    通过在父组件中绑定上面传递来的事件名来处理数据

    例如上面传递来的是this.$emit('func','传递值')

    那么父组件就要@func='a事件'

    再在父组件的methods中处理a事件(data)

    <body>
		<div id="app">
			{{message}}
			<cpn @itemclick='cpnclick'></cpn>
		</div>
		
		<template id='myCpn'>
			<div>
			<h2>{{cmessage}}</h2>
			<button @click="btnclick">点击按钮 传递值</button>
			</div>
		</template>
	</body>
	<script>
		const cpn = {
			template:'#myCpn',
			data(){
				return{
					cmessage:'子组件的message',
				}
			},
			methods:{
				btnclick(){
					console.log('发送事件',this.cmessage)
					this.$emit('itemclick',this.cmessage)
				}
			}
		}
		
		let app = new Vue({
			el : '#app',
			data : {
				message:`父亲的message`
			},
			methods:{
				 cpnclick(data){
				   console.log('接受事件与数值')
				   this.message = data
				 }
			},
			components:{
				cpn
			}
		})
	</script>

const { VueLoaderPlugin } = require('vue-loader')

plugins: [
    new VueLoaderPlugin()
  ]
