    # meta:

    用于设置网页中的元数据，用于搜索引擎

    <!-- charset:字符集 
     name：数据名称
             keywords:网站关键字
             description:网站的描述内容，显示在搜索引擎里
     content：数据内容
     http-equiv:对网页进行重定向,content内用于写时间和地址
       -->
<meta name="keywords" content="网上购物，网购，手机">
<meta name="description" content="京东巴拉巴拉.........">
<meta http-equiv="refresh" content="3;url=http://www.baidu.com">

    # 网站图标：

    <link rel="icon" href="./favicon.ico">
一般网站logo都存储在网站的根目录下，名字定义为favicon.ico

    # 语义化标签：

    关注标签的语义而不是关注标签的样子，例如h1的重要性仅次于title，搜索引擎会先从title查看再查看h1,一个页面一般只有一个h1。

    # 块元素：

    独占一行的元素 block element，常网页进行宏观布局的。

    <blockquote>这个给标签用于长引用</blockquote>
    1

    # 行内元素：

    不独占一行的元素 例如em用于表示重点

    <q>这个标签是用于短引用</q>
    1

    # 标题分组：

    hgroup将有联系的标题进行分组

    <hgroup>
    <h1>回乡偶书</h1>
    <h2>其一</h2>
</hgroup>

    # 相关路径：

    用于服务器内页面的跳转

    <a href="./test.html">链接</a>   <!-- ./表示当前目录  -->
<a href="../test.html">链接</a>  <!-- ../表示上层目录 -->

    # 超链接：

    <a target="_blank"></a>          <!-- 用于打开新页面 -->
<a href="#"></a>                 <!-- 跳转到顶部 -->
<a href="#id"></a>               <!-- 利用#id来跳转到指定元素的位置 -->

    # 视频与音频：

    <audio controls>
    <!-- controls属性可以让用户控制播放 -->
    <source src="./xxx.mp3">   
     <!-- 正常情况下的用法 -->
    <embed src="./xxx.mp3" type="audio/mp3">   
     <!-- 这个大多数用于兼容ie8 -->
</audio>
<!-- 视频与音频大多相同，文件一般基本不存在服务器，一般通过使用云来存放-->
<video controls>
     <source src="./xxx.mp4">
     <embed src="./xxx.mp4" type="video/mp4">

    # 移动端网页

    每一款移动设备都有一个最佳的像素比

    一般我们只需要将像素比设置为该值即可得到最佳效果

    将像素比设置为最佳像素比的视口大小我们称其为完美试图

    完美视口
<meta name="viewport" 
      content="width=device-width,initial-scale=1.0">
